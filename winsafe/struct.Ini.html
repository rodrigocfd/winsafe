Struct winsafe::Ini

source · 
pub struct Ini {
    pub sections: Vec<IniSection>,
}
Available on crate feature kernel only.
Expand description

High-level abstraction to load, manage and serialize sections and key/value pairs of a .ini file.

Examples

Printing all sections, keys and values:

use winsafe::{self as w, prelude::*};

let ini = w::Ini::parse_from_file("C:\\Temp\\foo.ini")?;

for section in ini.sections.iter() {
    println!("Section: {}", section.name);
    for entry in section.entries.iter() {
        println!("Key: {}; Value: {}", entry.key, entry.val);
    }
}

Reading a value:

use winsafe::{self as w, prelude::*};

let ini = w::Ini::parse_from_file("C:\\Temp\\foo.ini")?;

println!("{}", ini.value("the_section", "the_key").unwrap());
