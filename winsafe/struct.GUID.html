Expand description

GUID struct.

The Default implementation returns GUID::NULL (all zeros).

Implementations§

source§

impl GUID

source

pub const fn new(guid_str: &str) -> Self

Creates a new GUID from a representative hex string, which can be copied straight from standard GUID declarations.

Panics

Panics if the string has an invalid format.

Examples
use winsafe::{self as w, prelude::*};

let g = w::GUID::new("43826d1e-e718-42ee-bc55-a1e261c37bfe");
