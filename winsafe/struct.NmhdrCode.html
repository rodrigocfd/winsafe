Struct NmhdrCode

source 
pub struct NmhdrCode(/* private fields */);
Available on crate feature comctl only.
Expand description

Notification code returned in NMHDR struct. This code is convertible to/from the specific common control notification codes – LVN, TVN, etc.

§Examples

use winsafe::{self as w, prelude::*, co};

// Convert LVN to NmhrCode:
let code = w::NmhdrCode::from(co::LVN::ITEMCHANGED);

// Convert NmhrCode to LVN – fails it code is not a valid LVN:
let lvn = co::LVN::try_from(code)?;
