pub struct BSTR(/* private fields */);

Expand description

A string data type used with COM automation.

Automatically calls SysFreeString when the object goes out of scope.

pub const fn leak(&mut self) -> *mut u16

Ejects the underlying LPWSTR pointer leaving a null pointer in its place, so that SysFreeString won’t be called.

Be sure to free the pointer, otherwise, as the name of this method implies, you will cause a memory leak.
