Struct CDERR

pub struct CDERR(/* private fields */);

Expand description

Common dialog box error codes.

Also includes PDERR, CFERR, FNERR and FRERR prefixes.

Implements the standard Error trait.

Does not implement SystemError because FormatMessage function does not offer support for it, so there is no way to obtain a textual description of the error codes.
