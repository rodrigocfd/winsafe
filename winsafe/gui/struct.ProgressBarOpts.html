Expand description

Options to create a ProgressBar programmatically with ProgressBar::new.

Fields§

§position: (i32, i32)

Left and top position coordinates of control within parent’s client area, to be created.

Defaults to gui::dpi(0, 0).

§size: (i32, i32)

Width and height of control to be created.

Defaults to gui::dpi(120, 23).

§control_style: PBS

Progress bar styles to be created.

Defaults to PBS::SMOOTH.

§window_style: WS

Window styles to be created.

Defaults to WS::CHILD | WS::VISIBLE.

§window_ex_style: WS_EX

Extended window styles to be created.

Defaults to WS_EX::LEFT.

§ctrl_id: u16

The control ID.

Defaults to an auto-generated ID.

§resize_behavior: (Horz, Vert)

Horizontal and vertical behavior of the control when the parent window is resized.

Defaults to (gui::Horz::None, gui::Vert::None).

§range: (u32, u32)

Initial range, min and max values.

Defaults to (0, 100).

§value: u32

Initial progress value.

Defaults to 0.

§state: PBST

Initial state (normal, error or paused).

Defaults to PBST::NORMAL.
