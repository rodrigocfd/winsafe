pub struct CheckBox(/* private fields */);

Expand description

Native check box control, actually a variation of the ordinary Button: just a button with a specific style.

pub fn new(parent: &impl GuiParent, opts: CheckBoxOpts) -> Self

Instantiates a new CheckBox object, to be created on the parent window with HWND::CreateWindowEx.

§Panics

Panics if the parent window was already created – that is, you cannot dynamically create a CheckBox in an event closure.
