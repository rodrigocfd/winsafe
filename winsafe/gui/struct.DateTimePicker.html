pub fn new(parent: &impl GuiParent, opts: DateTimePickerOpts) -> Self

Instantiates a new DateTimePicker object, to be created on the parent window with HWND::CreateWindowEx.

§Panics

Panics if the parent window was already created – that is, you cannot dynamically create a DateTimePicker in an event closure.
