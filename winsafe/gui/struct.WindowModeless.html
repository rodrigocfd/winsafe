Struct WindowModeless

pub struct WindowModeless(/* private fields */);

Expand description

An user modeless window, which can handle events. Can be programmatically created or load a dialog resource from a .res file.

pub fn close(&self)

Closes the window by posting a WM_CLOSE message. This is the safest way to close any popup window, because you’ll able to process the wm_close event, just like if the user clicked the window “X” button.
