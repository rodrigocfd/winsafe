pub fn on_subclass(&self) -> &WindowEvents

Exposes the subclass events. If at least one event exists, the control will be subclassed.

Note: Subclassing may impact performance, use with care.

§Panics

pub fn new(parent: &(impl GuiParent + 'static), opts: TabOpts) -> Self

Instantiates a new Tab object, to be created on the parent window with HWND::CreateWindowEx.

§Panics

Panics if the parent window was already created – that is, you cannot dynamically create a Tab in an event closure.
