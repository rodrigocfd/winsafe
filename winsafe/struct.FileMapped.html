Expand description

Manages an HFILEMAP handle, which provides memory-mapped file operations, including read/write through slices. It is closed automatically when the object goes out of scope.

§Examples

Usually the fastest way to read a file as string is by memory-mapping it, like:

use winsafe::{self as w, prelude::*};

let f = w::FileMapped::open(
    "C:\\Temp\\foo.txt",
    w::FileAccess::ExistingReadOnly,
)?;
let raw_bytes = f.as_slice();
let text = w::WString::parse(raw_bytes)?.to_string();
