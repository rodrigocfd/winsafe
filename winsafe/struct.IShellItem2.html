Struct IShellItem2

pub struct IShellItem2(/* private fields */);

Expand description

IShellItem2 COM interface.

Automatically calls IUnknown::Release when the object goes out of scope.

Usually created with SHCreateItemFromParsingName function.

§Examples

use winsafe::{self as w, prelude::*};

let shi = w::SHCreateItemFromParsingName::<w::IShellItem2>(
    "C:\\Temp\\foo.txt",
    None::<&w::IBindCtx>,
)?;
