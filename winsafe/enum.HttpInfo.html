pub const fn unwrap_number(&self) -> u32

If the value is HttpInfo::Number, returns it; otherwise panics.

This is a syntactic sugar method to be used when you are sure of the HttpInfo content, its general use is discouraged.

§Panics

pub const fn unwrap_time(&self) -> SYSTEMTIME

If the value is HttpInfo::Time, returns a copy of it; otherwise panics.

This is a syntactic sugar method to be used when you are sure of the HttpInfo content, its general use is discouraged.
