Variants§

§

AliasAsync

Plays the system sound of the given co::SND_ALIAS asynchronously.

Fields

§alias: SND_ALIAS

System sound alias.

§stop: bool

If true, stops any sound currently playing asynchronously.

If set to false and a sound is currently playing, PlaySound will return an error.

§sentry: bool

Calls SoundSentry to give a visual cue as an accessibility feature.

§loops: bool

Plays the sound repeatedly until you call PlaySound with Snd::Stop.

MemSync

Reads the sound from memory and plays it synchronously.

Since PlaySound doesn’t provide a mechanism to notify its exit, there is no safe way to play from memory asynchronously.

Fields
