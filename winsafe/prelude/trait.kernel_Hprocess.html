Expand description

This trait is enabled with the kernel feature, and provides methods for HPROCESS.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

fn GetProcessTimes(&self) -> SysResult<(FILETIME, FILETIME, FILETIME, FILETIME)>

GetProcessTimes function.

Returns, respectively:

  1. creation time;
  2. exit time;
  3. kernel time;
  4. user time.
§Examples
use winsafe::{self as w, prelude::*, co};

let hprocess: w::HPROCESS; // initialized somewhere

let (creation, exit, kernel, user) = hprocess.GetProcessTimes()?;
