Available on crate feature gui only.
Expand description

This trait is enabled with the gui feature, and exposes tree view control notifications.

These event methods are just proxies to the GuiEventsParent of the parent window, who is the real responsible for the child event handling.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

ยงExamples
use winsafe::{self as w, prelude::*, gui, co};

let wnd: gui::WindowMain; // initialized somewhere
let tree: gui::TreeView;

tree.on().tvn_sel_changed(
    move |p: &w::NMTREEVIEW| -> w::AnyResult<()> {
        println!(
            "Old item: {} - new item: {} - action: {}",
            p.itemOld.pszText().unwrap(),
            p.itemNew.pszText().unwrap(),
            unsafe { co::TVC::from_raw(p.action) },
        );
        Ok(())
    },
);
