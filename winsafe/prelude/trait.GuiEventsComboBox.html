Available on crate feature gui only.
Expand description

This trait is enabled with the gui feature, and exposes combo box control notifications.

These event methods are just proxies to the GuiEventsParent of the parent window, who is the real responsible for the child event handling.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

ยงExamples
use winsafe::{self as w, prelude::*, gui};

let wnd: gui::WindowMain; // initialized somewhere
let cmb: gui::ComboBox;

let cmb2 = cmb.clone(); // to pass into the closure

cmb.on().cbn_sel_change(
    move || -> w::AnyResult<()> {
        if let Some(sel_text) = cmb2.items().selected_text()? {
            println!("New selected text: {}", sel_text);
        }
        Ok(())
    },
);
