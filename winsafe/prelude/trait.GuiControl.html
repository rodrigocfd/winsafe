Required Methods§

Source

fn ctrl_id(&self) -> u16

Returns the control ID, which is defined at control creation.

The control ID should be unique within a parent.

Provided Methods§

Source

fn focus(&self) -> SysResult<()>

In an ordinary window, simply calls HWND:SetFocus.

In a dialog window, sends a wm::NextDlgCtl message. This is preferable to the HWND::SetFocus because it takes care of border highlighting, like the native Button control needs.
