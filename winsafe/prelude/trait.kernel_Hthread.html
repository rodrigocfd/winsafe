Available on crate feature kernel only.
Expand description

This trait is enabled with the kernel feature, and provides methods for HTHREAD.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

fn GetThreadTimes(&self) -> SysResult<(FILETIME, FILETIME, FILETIME, FILETIME)>

GetThreadTimes function.

Returns, respectively:

  1. creation time;
  2. exit time;
  3. kernel time;
  4. user time.
§Examples
use winsafe::{self as w, prelude::*, co};

let hthread: w::HTHREAD; // initialized somewhere

let (creation, exit, kernel, user) = hthread.GetThreadTimes()?;
