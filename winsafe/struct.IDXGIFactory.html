Expand description

IDXGIFactory COM interface over IDXGIFactoryVT.

Automatically calls Release when the object goes out of scope.

Usually created with CreateDXGIFactory function.

§Examples

use winsafe::{self as w, prelude::*};

let factory = w::CreateDXGIFactory()?;
