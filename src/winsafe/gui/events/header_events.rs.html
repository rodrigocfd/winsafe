5/// This trait is enabled with the `gui` feature, and exposes header control

7///
8/// These event methods are just proxies to the
9/// [`GuiEventsParent`](crate::prelude::GuiEventsParent) of the parent window,
10/// who is the real responsible for the child event handling.
11///
12/// Prefer importing this trait through the prelude:

14/// ```no_run
15/// use winsafe::prelude::*;
16/// ```
